You want to learn a language, you need to start from the alphabet. Norwegian uses a modified Latin alphabet, with additions like æ, ø, å. For pronounciation, we need to get familiar with individual letters.

Pronounciation is mainly rot memorization but after alphabet, there are couple of basic rules that can guide us. Let's start with vowels.

If a vowel is followed by two consonants, it is a short vowel. If it is followed by one consonant, then it is a long vowel.

Pronounciation is contextual, unlike Turkish. A word can be pronounced different depending on the context. Thus, it requires memorization afterall. For example, we did not prononunce "d" at the end of råd above. But in the word rådhuset which means the town hall, the council hall; it is pronounced as {roodhüse'}.

Phrases :
Gratulerer med dagen {gratülerir me'dagin}
Literally "congratulations with the day", a phrase for [happy birthday]

Ta det med ro {taa de me' ruu}
Literally meaning "take it with calmness", a phrase for [take it easy]
For example, "å finne roen" means "to find the calmness" or "to find the peace".
